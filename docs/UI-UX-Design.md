# UI/UX Design Specification
**Project:** PT Mahaga Widya Cita Corporate Website & Portal

## 1. Brand Identity
- **Logo:** High Definition (1000x1000px), Transparent PNG.
- **Primary Color:** Deep Blue (`#1E6FD9`) - Represents trust, intelligence, and professionalism.
- **Secondary Colors:** 
  - Indigo/Purple for gradients (`#6366f1`)
  - Accent Gold/Yellow (`#F59E0B`) for highlights.
- **Typography:** 'Plus Jakarta Sans' (Primary font for headings and body). Clean, modern, and highly legible.

## 2. Design Principles
- **Glassmorphism:** Use of translucent backgrounds with background-blur for navbar and cards.
- **Micro-interactions:** Buttons and cards should scale up slightly on hover (`scale-105`) with smooth transitions (`duration-300`).
- **Whitespace:** Ample padding and margins to create a breathable, premium feel.

## 3. Responsive Breakpoints
- Mobile: `< 768px` (Stack layouts, hamburger menu)
- Tablet: `768px - 1024px`
- Desktop: `> 1024px` (Full grid layouts, expansive hero sections)
