# User Stories
**Project:** PT Mahaga Widya Cita Corporate Website & Portal

## 1. As a General Visitor
- I want to see the company's profile and services on the homepage so that I can quickly understand what PT Mahaga Widya Cita does.
- I want to read the latest articles and policy reviews so that I can stay updated with the company's insights.
- I want to subscribe to the newsletter so that I can receive periodic updates.
- I want to fill out a contact form so that I can inquire about services.

## 2. As a Corporate Client
- I want to view detailed service pages (e.g., Smart Consulting, Governance Review) so that I can evaluate if the company meets my business needs.
- I want to contact the sales or consulting team directly through the platform.

## 3. As an Admin/Editor
- I want to log into a secure CMS dashboard to manage content.
- I want to create, edit, and publish articles and policy reviews.
- I want to manage newsletter subscribers.
- I want to upload and manage media files securely.
